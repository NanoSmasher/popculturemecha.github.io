I like lists.  Here's another one.

I've compiled a list of manga that I want to read, but can't for a number of reasons.  If it hasn't been scanlated/translated/licensed in English, if it isn't finished, if it was cancelled etc., then I won't (be able) to read it.  However, these reasons don't necessarily make it completely impossible for me to one day be able to pick these up, and so they go on this wishlist.  And yes, cancelled manga can sometimes be brought back by the original authors, but it's rare.  I'm just saying.

Anyways, here we go!  (I've linked as many as I could with their MangaUpdates page, so that you can bug people to start scanlating, join scanlating groups, or find where the licensed versions can be sold near you.  And yes, an anime version will be come up some time in the future)

"Fushigi" Toriatsukaimasu: Tsukumodou Kottouten
